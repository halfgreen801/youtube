아이폰 앱 설치 방법

iPhone과 iPad에서는 브라우저가 설치 창을 자동으로 띄울 수 없습니다. Safari에서 직접 홈 화면에 추가해야 합니다.

  1. Safari에서 이 페이지를 엽니다.
  2. 하단 또는 상단의 공유 버튼을 누릅니다.
  3. 홈 화면에 추가를 선택합니다.
  4. 오른쪽 위 추가를 누르면 앱처럼 실행할 수 있습니다.

앱 설치는 홈 화면에 바로가기를 추가하는 기능입니다. 저장 목록 동기화는 내보내기/가져오기 또는 로그인 동기화를 사용하세요.
